**Solutions when position is specified:**

As we can see from the diagram, $\phi=\theta_1+\theta_3-\frac{\pi}{2}$.

$x=d_2cos(\theta_1)+a_3cos(\phi)=d_2cos(\theta_1)+a_3cos(\theta_1+\theta_3-\frac{\pi}{2})=d_2cos(\theta_1)+a_3sin(\theta_1+\theta_3)​$

$y=d_2sin(\theta_1)+a_3sin(\phi)=d_2sin(\theta_1)+a_3sin(\theta_1+\theta_3-\frac{\pi}{2})=d_2sin(\theta_1)-a_3cos(\theta_1+\theta_3)$

Given only an arbitrary desired position of the end effector, there will be two equations and three unknown variables. Therefore there will be infinite solutions.

**Assumptions about the desired position:**

$x^2+y^2=d_2^2+a_3^2+2d_2cos(\theta_1)a_3sin(\theta_1+\theta_3)-2d_2sin(\theta_1)a_3cos(\theta_1+\theta_3)=d_2^2+a_3^2+2d_2a_3sin(\theta_3)$

Since $-1\leq sin(\theta_3)\leq 1$, $d_2^2+a_3^2-2d_2a_3\leq x^2+y^2\leq d_2^2+a_3^2+2d_2a_3$.

This means $(d_2-a_3)^2\leq x^2+y^2\leq (d_2+a_3)^2$. So we have to assume that the desired position is within $(d_2+a_3)$ distance from origin $O_0$, i.e. the first joint, but not within $(d_2-a_3)$ distance from $O_0$. In other words, the desired position should be within the robot's workspace.

**Solutions when position and orientation are both specified:**

$d_2cos(\theta_1)=x-a_3cos(\phi)​$

$d_2sin(\theta_1)=y-a_3sin(\phi)​$

$\theta_1=atan2(y-a_3sin(\phi), x-a_3cos(\phi))​$

$\theta_3=\phi-\theta_1+\frac{\pi}{2}​$

$d_2=\frac{x-a_3cos(\phi)}{cos(\theta_1)}​$
